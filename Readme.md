# Computer Vision

A repository containing computer vision examples.

## Prerequisites
The following prerequisites are required to run the examples in this repository:

1. Install [Python](https://www.python.org/downloads)
2. Install [uv](https://docs.astral.sh/uv/getting-started/installation/)
3. Install [Git](https://git-scm.com/downloads)
4. Install a code editor (e.g. [Visual Studio Code](https://code.visualstudio.com/Download))

## Installation

To install Computer-Vision using [`uv`](https://docs.astral.sh/uv/getting-started/installation/), follow these steps:

1. Clone the Computer-Vision repository:

    ```bash
    git clone https://github.com/Deltares-research/Computer-Vision.git c:\...\Computer-Vision
    ```

2. Navigate to the cloned repository:

    ```bash
    cd c:\...\Computer-Vision
    ```

3. Synchronize the virtual environment:
    ```bash
    uv sync
    ```

4. Request the `.env` file from the repository maintainer and place it in the root of the cloned repository.

## Usage
To use the workflows in this repository, open the Jupyter notebooks in the `examples` directory using your code editor or Jupyter Notebook interface. Follow the instructions within each notebook to run the examples.
